# 02-Exploratory-Data-Analysis

Notebooks focused on understanding and analyzing data through visualizations and statistics.

## Notebooks

- **19-understanding-data.ipynb** - Data understanding framework (shape, types, missing values, describe)
- **20-eda-univariate.ipynb** - Analyzing single variables with count plots and distributions
- **21-eda-multivariate.ipynb** - Exploring relationships between multiple variables
- **22-pandas-profiling.ipynb** - Automated EDA with pandas profiling library

## Skills Learned
- Data shape and structure analysis
- Missing value identification
- Statistical summaries
- Univariate and multivariate visualizations
- Automated report generation

## Datasets Used
- Titanic (Passenger survival)
- Tips (Restaurant tipping)
- Flights (Time series)
