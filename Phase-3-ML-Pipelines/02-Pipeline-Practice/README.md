# 02-Pipeline-Practice

Hands-on practice building ML pipelines with real-world datasets.

## Notebooks

- **29-03-practice-pipeline.ipynb** - Telecom Churn prediction pipeline
- **29-04-practice-pipeline.ipynb** - Loan Default prediction pipeline

## Datasets

### Telecom Churn
- **Rows:** 15,000
- **Features:** gender, age, city, plan, tenure_months, monthly_bill, support_calls, data_gb, payment_method, has_internet
- **Target:** churn (Yes/No)
- **Challenges:** Missing values, dirty categories

### Loan Default
- **Rows:** 15,000
- **Features:** gender, age, education, job_type, married, experience_years, monthly_income, loan_amount, credit_score
- **Target:** default (Yes/No)
- **Challenges:** Missing values, dirty categories, mixed data types

## Skills Practiced

- Building end-to-end pipelines
- Handling missing values within pipelines
- Dealing with dirty/unknown categories
- Working with multiple categorical features
